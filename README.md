# Homebrew tap for **open-s4c** tools and libraries.

### Available Formulae

---

### **1. tmplr**

Simple template replacement tool for C projects.

**Install:**

```sh
brew tap open-s4c/tap
brew install tmplr
```

After installation, verify it with:

```sh
tmplr -V
```

### **2. vatomic**

vatomic is a header library of atomics operations, supporting mainstream architectures: ARMv7, ARMv8 (AArch32 and AArch64), RISC-V, and x86_64.

**Install:**

```sh
brew tap open-s4c/tap
brew install vatomic
```

After installation, verify it with:

```sh
brew test vatomic
```
